# hiptty TUI 设计文档

**版本**: 1.0
**日期**: 2026-07-01
**定位**: hiptty 终端客户端 UI/UX 权威参考，所有 TUI 开发以此为准。

---

## 1. 设计原则

1. **无底色**。TUI 不绘制背景色块，完全融入终端原生背景。阅读体验由前景色层次区分，不由色块划分。
2. **内容优先**。弱化边框/线条，用间距、缩进、颜色表达层级。装饰最小化，信息最大化。
3. **图片先行**。面向现代终端，avatar、表情、帖子图片一律通过 `ratatui-image` 协议渲染（Kitty/iTerm2/Sixel）。不做降级讨论。
4. **键盘第一，鼠标辅助**。所有操作有键盘快捷键，鼠标点击作为同等操作的支持。
5. **瀑布流**。列表不显式分页，滚动接近底部时自动加载下一页。
6. **动画过渡**。页面切换、弹层进出、列表滚动、Logo 呼吸均带平滑动画。

## 2. 视觉规范

### 2.1 色板

不设背景色。语义色分 Dark/Light 两套，由用户设置切换。

| Token | Dark | Light | 用途 |
|-------|------|-------|------|
| `primary` | `#d4d4d8` | `#27272a` | 正文 |
| `secondary` | `#a1a1aa` | `#52525b` | 次要文字（时间、元信息） |
| `accent` | `#818cf8` | `#6366f1` | 选中高亮、导航竖线、链接 |
| `warn` | `#fbbf24` | `#d97706` | 警告 |
| `error` | `#f87171` | `#dc2626` | 错误、删除操作 |
| `quote_bg` | `#3f3f46` | `#e4e4e7` | 引用块底色（可选） |
| `dim` | `#71717a` | `#a1a1aa` | 分隔线、非活跃元素、placeholder |
| `title_color` | 论坛原色 | 论坛原色 | 帖子标题颜色，解析自 HTML，**始终保留** |

### 2.2 Logo

**登录页** — 2 行 block-char，居中：

```
█░█ █  █▀█ █▀▄ █▀█      ← "HI" 紫色 #a78bfa, "PDA" 浅灰 #9ca3af
█▀█ █  █▀▀ █▄▀ █▀█      ← 宽 ~19 字符
```

**Title Bar** — 1 行纯文本 `HIPDA`，紫色↔灰色渐变动画过渡（每帧微调 hue，实现呼吸/流动感）。

### 2.3 字体 & 图标

- 主文字：等宽字体
- 图标：**全站使用 Nerd Font**，不使用 emoji
- Nerd Font 图标参考表：

| 含义 | 图标 | Nerd Font codepoint | 用途 |
|------|------|---------------------|------|
| 置顶 |  | `nf-fa-thumb_tack` `U+F08D` | 帖子列表 sticky 标记 |
| 热门 |  | `nf-mdi-fire` `U+F502` | 帖子列表热门标记 |
| 带图 |  | `nf-fa-picture_o` `U+F03E` | 帖子列表含图标记 |
| 投票 |  | `nf-mdi-vote` `U+F681` | 帖子列表投票标记 |
| 新帖 | NEW | 纯文本 | 帖子列表新帖标记，`accent` 色 |
| 回复数 |  | `nf-fa-comments` `U+F27A` | 回复计数 |
| 查看数 |  | `nf-fa-eye` `U+F06E` | 查看计数 |
| 通知 |  | `nf-fa-bell` `U+F0A2` | Title Bar 未读通知 |
| 私信 |  | `nf-fa-envelope` `U+F0E0` | Title Bar 未读私信 |
| 附件 |  | `nf-fa-paperclip` `U+F0C6` | 帖子附件标记 |
| 在线 |  | `nf-fa-check` `U+F00C` | 用户在线状态 |

### 2.4 间距约定

- 列表项之间：1 空行
- 楼层之间：dim 横线 + 1 空行
- 正文段落之间：1 空行
- 引用块缩进：2 字符
- Title Bar 与内容分隔：dim 横线

- 登录成功后带淡入过渡动画进入 ThreadFeed（默认版块 Discovery `fid=2`）

### 5.1 Title Bar

| Row | 左 | 右 |
|-----|-----|-----|
| 1 | Logo `HIPDA` (紫色↔灰色动画) | `用户名` ``(仅有通知时) ``(仅有私信时) |
| 2 | 当前版块名（取自 `Forum.name`，如 `Discovery`） | — |

### 5.2 列表项

每个项 2 行高 × 全宽。

**左列**（~8 cols）：
- avatar 图片，2 行高 × 3~4 字符宽（由 `ratatui-image` 渲染）

**右列第一行**（剩余宽度）：
- `帖子标题`（使用 `title_color`，超长截断）
- 类型图标（右对齐）：``置顶 ``热门 ``带图 ``投票 `NEW`新帖 （Nerd Font）

**右列第二行**：
- `作者 · thread_type`（`thread_type` 为帖子分类，如 评测/二手/求助，仅在存在时显示）
- `回复数  查看数`（Nerd Font 图标）
- `相对时间 · 最后回复人`（最后回复人仅在有回复时显示）

### 5.3 自动加载

滚动到距底部 5 项以内时，自动请求 `threads list --fid N --page next`，追加到列表末尾。加载中在底部显示 `── 正在加载... ──`。新数据以滑入动画追加。

### 5.4 页面切换动画

- 从 ThreadFeed → ThreadDetail：详情页从右侧滑入（slide-left）
- 从 ThreadDetail → ThreadFeed：列表页从左侧滑入（slide-right）
- 弹层：从底部或中央 fade-in + scale-up
- 弹层关闭：fade-out + scale-down

### 5.5 数据映射

```
ThreadSummary
  tid           → Enter 跳转目标
  title         → 标题行
  title_color   → 标题颜色
  author        → 第二行作者
  avatar_url    → avatar 图片
  thread_type   → 帖子分类（第二行，作者旁）
  reply_count   → N
  view_count    → N
  time_update   → 相对时间
  last_post     → 最后回复人
  sticky        → 
  with_pic      → 
  is_new        → NEW (纯文本, accent 色)
  is_poll       → 
```

### 6.1 Title Bar

| Row | 左 | 右 |
|-----|-----|-----|
| 1 | Logo `HIPDA` | `用户名` ``(仅未读时) ``(仅未读时) |
| 2 | `版块名 > 帖子标题`（超长截断） | `回复数  查看数` |

### 6.2 楼层结构

每个楼层：
- **导航竖线** `│`：左侧 1 列。当前 active 楼层为 `accent` 色，其他为 `dim`。键盘/鼠标切换楼层时竖线颜色平滑过渡。
- **楼层分隔**：dim 横线，无文字
- **顶部信息行**：`avatar(2行高) + 用户名 + 楼层号 + 时间`
- **签名**：截断到 25 字符，显示在 avatar 右侧第二行（与时间同行），`dim` 色 + 引号包裹
- **正文**：avatar 下方开始，全宽

### 6.3 内容渲染规则

按 `ContentNode` 类型依次渲染：

| ContentNode | 渲染方式 |
|-------------|---------|
| `Text { spans }` | 段落。`ContentSpan::Text` 带 `Style`（bold/italic/color）。`ContentSpan::Smiley` 内联渲染 smiley 图片。超宽自动换行。 |
| `Quote { author, time, text, pid, tid }` | 左竖线 `┃`（`dim` 色）+ 2 字符缩进 + `@author 说:` （`accent` 色）+ 正文。如有 pid/tid 可被 Enter 点击跳转。 |
| `Image { url, thumb_url, size }` | 通过 `ratatui-image` 异步加载并内联渲染。所有图片直接显示。 |
| `Attachment { name, url, size }` | ` name (size)` 形式展示（`` Nerd Font 附件图标）。按 `o` 用外部程序打开。 |
| `FloorRef { floor, author, pid, tid }` | `>>> #floor @author` （`accent` 色），可被 Enter 点击跳转到对应楼层。 |
| `AppMark { text, url }` | `▸ text`（`dim` 色），按 `o` 打开 url。 |

### 6.4 投票

若 `Post.poll` 存在（仅首帖），在正文前渲染：

```
  ┌─  投票: 你用什么牌子的显示器？ ─────────────────────┐
  │                                                        │
  │  ●  Dell         ████████░░  42 票 (68.5%)             │
  │     LG           ████░░░░░░  22 票 (35.5%)             │
  │     Samsung      ███░░░░░░░  15 票 (24.2%)             │
  │                                                        │
  │  最多可选 2 项                                         │
  └────────────────────────────────────────────────────────┘
```

- 投票条形图用 Unicode block `█` + `░` 构建
- 当前选中项用 `accent` 色高亮
- 投票投递功能预留，当前只读

### 6.5 数据映射

```
ThreadDetail
  tid, title, fid          → Title Bar
  posts[]                  → 楼层列表
    Post
      pid, floor, author   → 楼层头部
      uid                  → 用户标识 (点击可发私信)
      avatar_url           → avatar 图片
      time                 → 相对时间
      content[]            → 正文 ContentNode 列表
      poll                 → 投票表格
      signature            → 截断签名
      warned               → 风险提示标记
```

## 11. 弹层体系

所有弹层带 fade-in + scale-up 动画（约 150ms），关闭时 fade-out + scale-down。

### 11.1 版块切换 (ForumPicker)

`f` 键触发，居中浮层：

```
  ┌─ 切换版块 ──────────────────────────┐
  │                                      │
  │  默认版块                             │
  │  ──────────                          │
  │  ●  Discovery                       │
  │     Geek Talks                      │
  │     Buy & Sell                      │
  │                                      │
  │  全部版块                             │
  │  ──────────                          │
  │     E-INK                           │
  │     Smartphone                      │
  │     iPhone, iPod Touch，iPad        │
  │     Android, Chrome, & Google       │
  │     ...                             │
  │                                      │
  │  j/k 移动  Enter 切换  Esc 关闭      │
  └──────────────────────────────────────┘
```

- 版块名直接取自 `Forum.name` 原始数据，不做翻译
- 默认版块区：显示用户在设置中选择的 3 个常驻版块
- 全部版块区：分隔线下方列出剩余所有版块

## 13. 快捷键 Scope 体系

| Scope | 适用场景 | 键位 |
|-------|---------|------|
| **Global** | 所有非编辑状态 | `Esc` 返回/关闭, `:` 命令, `?` 帮助 |
| **List** | ThreadFeed, Search, PMList, NotificationList, MyThreads, Favorites | `j/k` 上下, `Enter` 打开, `/` 搜索, `n` 新帖, `f` 切换版块, `g/G` 首页/末页, `b` 返回 |
| **Detail** | ThreadDetail, PMThread | 继承 List + `r` 回复, `q` 引用, `e` 编辑, `d` 删除, `v` 查看图片, `o` 打开链接/附件, `#` 跳楼层 |
| **Editor** | PostComposer | `Ctrl+S` 发送, `Esc` 取消, `Ctrl+I` 插入图片 |
| **Overlay** | ForumPicker, Help, Confirm, Settings | `j/k` 移动, `Enter`/`y` 确认, `n`/`Esc` 关闭 |

鼠标映射：点击列表项 = `Enter`；点击帖子操作区 = 对应快捷键。滚动 = `j/k` 等价。

## 16. 边界情况

| 场景 | 处理 |
|------|------|
| 帖子无标题颜色 | 使用 `primary` 色 |
| 帖子标题为空 | 显示 `(无标题)`，`dim` 色 |
| 图片加载失败 | 显示 `[图片加载失败]` `dim` 色，保留 `o` 打开链接 |
| 头像加载失败 | 显示用户名首字符或默认占位（`dim` 色边框方块） |
| 引用块嵌套 | 只渲染一层，嵌套引用展平 |
| 超长帖子（万层） | 虚拟滚动 + 懒加载分页 |
| 网络断开 | Toast 提示 "网络连接失败"，输入操作缓存或提示 |
| 登录凭证过期 | 自动弹出登录页 |
| 发帖节流（30s） | Toast 提示 "发帖过快，请 X 秒后重试" |
| GBK 编码用户名 | core 已处理编码，TUI 直接使用 |
| 通知/私信数超过 99 | 不显示数字，仅显示图标 |
| 终端窗口过小 | 最小尺寸 80×24，低于阈值显示警告 |
| `thread_type` 为空 | 第二行仅显示作者名，不显示 `·` 接分类 |
| 回复数/查看数为 0 | 列表第二行不显示对应图标和数字 |
